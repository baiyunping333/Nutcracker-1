Detailed Description

template<typename side>
class Nutcracker::StateSite< side >

State site tensors.

Each state site tensor is associated with a qudit,and has three ranks. Two of the three ranks correspond to its connections to its neighbors (i.e., its "left" and "right" dimensions) and the last rank corresponds to the state space of the qudit at the site (i.e., its "physical" dimension).

The type tag side on the class uses the type system to embed information about the normalization of the tensor in the type, so that for example a left-normalized state site tensor cannot be used in the place of a right-normalized tensor.
