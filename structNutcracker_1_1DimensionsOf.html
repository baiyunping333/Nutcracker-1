Detailed Description

template<typename T>
struct Nutcracker::DimensionsOf< T >

A parameter wrapper class that indicates a tensor should be constructed with the same dimensions as the wrapped tensor.

It is easiest to wrap a tensor in this class by using the convenience functions dimensionsOf.
